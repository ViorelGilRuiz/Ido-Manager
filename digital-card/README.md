# Digital Card

Abre `card.html` para mostrar tu tarjeta digital con QR en movil o escritorio.

## Contenido
- `assets/profile-viorel.jpg`: tu foto de perfil.
- `assets/viorel-contact-qr.png`: QR de contacto (GitHub + CV).
- `assets/ViorelGil_CV.pdf`: CV en PDF para publicar en GitHub.
- `viorel-contact.vcf`: tarjeta de contacto para importar en movil.

## Publicacion en GitHub
1. `git add digital-card`
2. `git commit -m "Add professional digital card with QR and CV"`
3. `git push origin main`
