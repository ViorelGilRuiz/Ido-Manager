## Featured project

## I Do Manager

**I Do Manager** is a modular SaaS platform for wedding planners and end clients. It is designed as a real digital product with a clear operational domain, not as a generic showcase project.

### Why this project is relevant for recruiters

- It demonstrates **full-stack execution with strong backend foundations**
- It shows product thinking, not only interface work
- It includes authentication, domain entities, CRUD flows, document logic and operational workflows
- It reflects a maintainable architecture using **Angular + NestJS + Prisma + PostgreSQL**
- It is built around a real business model with clear entities such as `User`, `Business`, `Event`, `Template` and `Document`

### Backend signals this project sends

- Strong service structure and domain separation
- Secure authentication flow with JWT and refresh token strategy
- Relational data model with scalable patterns
- API versioning and reusable backend foundations
- Product-ready logic instead of tutorial-level endpoints
